==== Step1 ====
8. Configure Nagios Client
(Installation - NRPE Client (CentOS 7.1) - How To,Gutzmann n.d.)
Open the nagios npre configuration file /etc/nagios/nrpe.cfg 
vi /etc/nagios/nrpe.cfg
Add the following to allow custom monitoring commands to be included in the /etc/nagios directory
allowed_hosts=127.0.0.1,nagios_1.yule.gov.net nagios_2.yule.gov.net
include_dir=/etc/nagios
Start and enable the appropriate services
systemctl enable chronyd
systemctl restart chronyd
systemctl enable nrpe
systemctl restart nrpe
systemctl enable xinetd
systemctl restart xinetd
