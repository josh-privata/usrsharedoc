==== Step1 ====
1. Configure Networking
Open the network configuration file located at /etc/sysconfig/network-scripts/ifcfg-enp0s8
vi /etc/sysconfig/network-scripts/ifcfg-enp0s8
And make the following changes to enable DHCP
HWADDR=00:00:00:00:00:00
TYPE=Ethernet
# Assign IP from DHCP
BOOTPROTO=dhcp
DEFROUTE=yes
PEERDNS=yes
PEERROUTES=yes
IPV4_FAILURE_FATAL=no
IPV6INIT=yes
IPV6_AUTOCONF=yes
IPV6_DEFROUTE=yes
IPV6_PEERDNS=yes
IPV6_PEERROUTES=yes
IPV6_FAILURE_FATAL=no
NAME=enp0s8
UUID=e5a5d8e9-b5d4-4b5e-bd1e-6ebcd0144dfa
# Enable interface at system boot
ONBOOT=yes

==== Step2 ====
1. Configure Networking
Open the network configuration file located at /etc/sysconfig/network-scripts/ifcfg-enp0s8
vi /etc/sysconfig/network-scripts/ifcfg-enp0s8
Add placeholder text that will be filled in after server provisioning.
HWADDR=00:00:00:00:00:00
TYPE=Ethernet
# Enabled Static IP Address
BOOTPROTO=none
DEFROUTE=yes
PEERDNS=yes
PEERROUTES=yes
IPV4_FAILURE_FATAL=no
IPV6INIT=yes
IPV6_AUTOCONF=yes
IPV6_DEFROUTE=yes
IPV6_PEERDNS=yes
IPV6_PEERROUTES=yes
IPV6_FAILURE_FATAL=no
NAME=enp0s8
UUID=e5a5d8e9-b5d4-4b5e-bd1e-6ebcd0144dfa
# Enable interface at system boot
ONBOOT=yes
# IP Address
IPADDR=IP-Address
# Subnet Mask
NETMASK=255.255.255.0
# Default Gateway
GATEWAY=IP-Gateway
# Primary DNS
DNS1=IP-DNS-Server1
# Secondary DNS
DNS2=IP-DNS-Server2
# Search Domain
DOMAIN=yule.gov.net
